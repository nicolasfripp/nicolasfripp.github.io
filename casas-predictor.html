Predicción de Precios de Casas en Uruguay

Modelo de Machine Learning para Estimar Valores Inmobiliarios

Ver código ↗

Objetivo del Proyecto

Desarrollar un modelo predictivo capaz de estimar el precio de venta de casas en Uruguay a partir de sus características. El modelo busca servir como herramienta de referencia para vendedores que necesitan fijar un precio, para compradores que quieren detectar oportunidades, y para analizar qué variables son las más determinantes en el valor de una propiedad.

Problema Abordado

El mercado inmobiliario uruguayo carece de herramientas accesibles para estimar precios de propiedades de forma objetiva. Se partió de una base de datos de más de 177,000 publicaciones de casas en venta extraídas mediante web scraping de Mercado Libre y El Gallito, abarcando el período 2023-2025. El desafío principal radicó en la calidad de los datos: duplicados, valores faltantes, errores de carga por parte de los usuarios (monedas incorrectas, valores ficticios como "1111111"), y la necesidad de unificar dos fuentes de datos con estructuras diferentes.

Decisiones Técnicas

  • Se unificaron dos fuentes de datos (Mercado Libre y El Gallito) con variables equivalentes pero en columnas distintas, mergeando NEIGHBORHOOD/ADDRESS_CITY_NAME, CONDITION/ITEM_CONDITION y las fechas de proceso
  • Se imputaron departamentos faltantes mediante mapeo por moda de barrio y prompt engineering con IA para los casos restantes, logrando pasar de 94% nulos a 0%
  • Se mantuvieron dos datasets paralelos: uno con agrupación de categorías para regresión lineal y otro preservando todas las categorías para CatBoost
  • Se eligió CatBoost como modelo principal dado que 6 de las 10 variables explicativas son categóricas, aprovechando su manejo nativo de este tipo de datos
  • Se entrenaron modelos segmentados por departamento (Montevideo, Maldonado, Canelones) para capturar dinámicas locales del mercado

Datos

+177,000 publicaciones de casas en venta en Uruguay (Mercado Libre y El Gallito), reducidas a ~37,000 observaciones tras limpieza. Variables finales: barrio, departamento, área cubierta, baños, dormitorios, garajes, condición, aire acondicionado, tour virtual, tipo de publicación y origen.

Transformación y Limpieza de datos

  1. Carga y unificación de dos fuentes de datos con variables equivalentes en columnas distintas
  2. Eliminación de variables irrelevantes (descripción, título, IDs, etc)
  3. Imputación de datos faltantes: departamentos por moda de barrio + IA, condición reclasificada en 4 categorías (nuevo, usado, en construcción, sin especificar)
  4. Conversión de precios a USD usando tasas de cambio mensuales y eliminación de registros con errores de moneda
  5. Limpieza de outliers: remoción de valores ficticios ("1111111"), precios menores a USD 1,000, y outliers por IQR
  6. Eliminación de duplicados y análisis exploratorio con visualizaciones univariadas y bivariadas
  7. Entrenamiento de modelos: Regresión Lineal (con transformaciones logarítmicas) y CatBoost (con optimización Optuna)

Resultados Obtenidos

Modelo MAPE Observaciones
Regresión Lineal (OLS) 39.9% Modelo base con limitaciones por la naturaleza categórica de los datos
CatBoost 33.8% Mejora significativa gracias al manejo nativo de variables categóricas
CatBoost + Optuna 32.0% Optimización de hiperparámetros con K-Fold (5 splits)
CatBoost + Optuna (Montevideo) 25.0% Mejor resultado — error típico de ± USD 50,000 sobre un precio promedio de USD 200,000
CatBoost + Optuna (Maldonado) 37.0% Modelo segmentado por departamento
CatBoost + Optuna (Canelones) 29.7% Modelo segmentado por departamento

La segmentación geográfica demostró que los modelos locales capturan mejor las dinámicas de cada mercado. El modelo final constituye una herramienta confiable para estimar valores inmobiliarios en el mercado uruguayo.
