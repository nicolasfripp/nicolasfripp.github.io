# House Price Prediction - Uruguay

A machine learning model that predicts house prices in Uruguay based on a dataset scraped from Mercado Libre and El Gallito (2023-2025). The model uses features such as location, covered area, number of rooms, bathrooms, condition, and more to estimate listing prices.

This project is useful for sellers looking for a pricing reference, for identifying over/undervalued properties, and for understanding which variables are most relevant when determining a property's price. The full analysis is in the `Casas.ipynb` notebook.
