Futbol Predictor

Predicción de Resultados de La Liga Española mediante Machine Learning

Objetivo del Proyecto

Construir un sistema de machine learning capaz de predecir resultados de partidos de La Liga española, generando probabilidades para tres posibles desenlaces: victoria local (H), empate (D) y victoria visitante (A). El sistema implementa dos escenarios de predicción: pre-partido y en vivo (medio tiempo).

Problema Abordado

La predicción de resultados deportivos es un problema de clasificación multiclase con alta incertidumbre inherente. Los partidos de fútbol están influenciados por múltiples factores difíciles de cuantificar. Este proyecto aborda el desafío combinando estadísticas históricas de 10 temporadas (2016-2025), datos de mercado de Transfermarkt y cuotas de 5 casas de apuestas para construir modelos predictivos robustos.

Decisiones Técnicas

  • Split temporal (temporadas 2016-2023 para entrenamiento, 2024-2025 para test) en lugar de aleatorio, respetando la naturaleza secuencial de los datos deportivos
  • Ventanas móviles de 5 partidos para calcular estadísticas rolling (goles, tiros, corners, tarjetas, puntos)
  • Class weights para manejar el desbalance de clases (los empates son menos frecuentes)
  • Optimización de hiperparámetros con Optuna: 50 trials con TPE sampler y 3-fold cross-validation
  • Modelado dual: escenario pre-partido (41 features) y escenario in-play con features adicionales de medio tiempo

Feature Engineering

41 features pre-match incluyendo rolling stats, valores de mercado, cuotas de apuestas y ratios de eficiencia. Features adicionales in-play: goles al medio tiempo, diferencia de tarjetas.

Fuentes de Datos

  • Estadísticas de partidos: 10 temporadas de La Liga (2016-2025)
  • Datos de Transfermarkt: valores de mercado de los equipos
  • Cuotas de 5 casas de apuestas: Bet365, Betwin, Pinnacle, William Hill y VCBet
  • Variables derivadas: probabilidades implícitas normalizadas, ratios de valor de mercado, diferencias de forma

Resultados Obtenidos

  • 6 modelos entrenados: 3 algoritmos (Random Forest, XGBoost, LightGBM) × 2 escenarios (pre-partido e in-play)
  • Métricas de evaluación: Accuracy, Log Loss, F1-Weighted y F1-Macro
  • Los modelos in-play mejoran significativamente la accuracy al incorporar datos de medio tiempo
  • Análisis completo de importancia de features para cada modelo
  • Matrices de confusión y reportes de clasificación detallados
  • Comparación visual exhaustiva entre modelos y escenarios
